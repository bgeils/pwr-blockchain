# PWR blockchain
Blockchain implementation of peer to peer energy marketplace

## Requirements/Setup

```
sudo npm install -g ethereumjs-testrpc
```
```
sudo npm install -g truffle
```


## To run
In one bash shell:
```
$ testrpc
```
In another bash shell:
```
$ cd ~/surge
$ truffle compile
$ truffle migrate
$ truffle build
$ truffle serve
```
navigate to localhost:8080
